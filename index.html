フェーズ別ROI 仕様モック

投資回収期間をベースに、店舗ライフサイクルのフェーズごとにROI表示を切り替える仕様のモックアップ。

ℹ️ これは仕様確認用のモックです。本番データは接続されていません。入力値はローカルに保存されません。

🏪 店舗ダッシュボード

単一店舗のフェーズ判定とROI表示
  • フェーズバッジ + 主ROI(フェーズ別)
  • 回収進捗バー / 予測回収月 / 回収倍率
  • フェーズ4段階のタイムライン表示
  • 坪月商 vs 業態ベンチマーク判定

🏢 会社ダッシュボード

全店舗を加重平均したROIとフェーズ別構成
  • 全社ROI = Σ(利益)÷Σ(投資) の加重平均
  • フェーズ別店舗数構成 / フェーズ別ROI
  • 店舗別フェーズ一覧テーブル

⚙️ 設定画面

フェーズモデル/境界倍率/業態ベンチマーク/店舗別設定
  • フェーズ段数(2〜6) + 各フェーズ閾値/色のカスタマイズ
  • 境界倍率(デフォルト1.5) 会社/店舗でオーバーライド
  • 業態マスタ(デフォルト7業態+顧客カスタム登録)
  • 店舗別: 業態/初期投資/オープン日/目標回収期間/ROI算出方式
